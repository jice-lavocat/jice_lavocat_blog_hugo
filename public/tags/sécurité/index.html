Modifier de façon récursive les droits des dossiers [Script Bash]

Vous êtes étudiants à l’Ecole Centrale Marseille? Vous êtes sur un réseau où les administrateurs se font du soucis pour la sécurité? Il y a fort à parier que la publication d’un site web présente quelques difficultés pour le néophyte. En effet, le serveur qui affichera vos pages web (apache?) se basera sur les droits que vous accordez à vos fichiers. Si ceux-ci sont mal réglés, vous n’aurez pas le droit d’y accéder par le web. »
